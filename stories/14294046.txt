Sip I/O is a lightweight sip proxy, location server, and registrar that provides a reliable and scalable SIP infrastructure for telephony carriers, communication service providers, and integrators. It also provides with capabilities that are suitable for the enterprise and personal needs. For a list of features and documentation about the project please visit the wiki. To get involve in the development of this project, please contact me at @sandedro.

Why Java 9? Sip I/O uses Oracle Nashorn Javascript Engine to take advantage of several ES6 features. These features are only available in latest and greatest version of Java.
