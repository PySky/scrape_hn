NectarJS is a cross-platform multi-purpose JavaScript compiler. You can write a JS program from your Android phone, and compile it for Linux or Windows, painless.

NectarJS is able to produce WebAssembly for your JavaScript code : no more need for obfuscation.

NectarJS is not finished yet and need more development.|||
