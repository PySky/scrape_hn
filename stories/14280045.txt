This is a nginx installer + configuration + modules + patches for web speed and security.

Feel free to use, add, modify at will. Apart the normal build dependencies, it will install (unless it's installed already), Google's compression engine (better for large files), and a number of modules. If you don't plan on using all the modules, see the install files to disable them.
