Apply custom CSS styling to your jupyter notebooks. Contributions are welcome!

You can always revert back to the default:

NOTE: Old ipython configuration files, for example at the default , may need to be removed in order for jupyter to use the correct files.

If no arguments are supplied, the program will revert the jupyter notebook style back to default.

Any combination of the style types can be specified, and the program will mix together the associated color/layout/typographic/code font/background css files accordingly, writing it to the file used by the notebook.
