The name of this domain is î.com; it is an Internationalized Domain Name (IDN). According to the Unicode standard, the name for the î character is "Latin small letter i with circumflex". One way to reach this domain is to type the punycode representation xn--fda.com in your browser bar. The Unicode code point for this character is U+00EE.

The character î (capital form: Î) is used in a number of languages, including Welsh, French, Romanian and Molvanîan. For more information, see the article on this character in Wikipedia.
