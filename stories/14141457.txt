Nissan Invalid Key ID on Push To Start



if you are getting invalid key id when starting.

1. unlock the door with mechanical key

2. open and close the drivers door 2 times

3. move the steering wheel to the left

4. push the button in harder

5. replace the battery in the key fob

6. bad door switch, replace the switch



if you have a technique that worked, please leave in the comments below|||
