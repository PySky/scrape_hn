To stay protected after support ends, you have some options. The first option is to upgrade your current PC. Very few older computers will be able to run Windows 10, which is the latest version of Windows. We recommend that you download and run the Windows Upgrade Assistant to check if your PC meets the system requirements for Windows 10 and then follow the steps in the tutorial to upgrade if your PC is able. For more detailed information, read the FAQ. You can also purchase a new PC. If your current PC can't run Windows 10, it might be time to consider shopping for a new one.
