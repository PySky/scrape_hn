JsonPack is a high-performance and extremely easy-to-use JSON serialization library for C++ 11, it provide a absolute abstraction layer of this Data Interchange Format. From now on, you only should take care about managing your C++ objects and JsonPack will make the rest.

Easy-to-use, contains a very short and intuitive API. See Example section Very fast: template metaprograming, zero string copy and fast number conversions. Support serialization/deserialization for c++ types: bool, char, int, unsigned int, long, unsigned long, long long,unsigned long long, float, double and std::string. Support serialization/deserialization for c++ standard associative containers: map, multimap, unordered_map and unordered_multimap with std::string key JSON keys match with C++ identifiers name convention when json blob read directly into the data object.
