(Short Message Service) is specified by the ETSI (standards GSM 03.40 and 03.38 ). It can contain up to 160 characters, when each character is written according the 7-bits GSM default alphabet.[ 7 bits default tabel ]



Next to a message the SMS containt also some meta-data, for example

 - Info about the senders ( Service center number, sender number) 

 - Protocol information  (Protocol identifier, Data coding scheme)

 - Time stamp 



There are two way to recieve and send SMS messages , PDU (protocol discription unit) and Text mode. In this document we focus on PDU mode.
