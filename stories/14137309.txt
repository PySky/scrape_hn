is small reverse websocket proxy designed for . It's purpose is to capture messages written to and received from Chrome Debugging Protocol, coalesce requests with responses, unpack messages from Target domain and provide easy to read, colored output. This tool is a fork of (and heavily inspired by) chromedp-proxy.

When using Headless Chrome navigate to inspectable pages and open inspector pane for url of your choosing. Then replace port in query param and point it to running instance (default port is 9223). Now you're able to see what Chrome Debugger is exactly doing. Enjoy!|||
