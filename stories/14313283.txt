The allows you to install the XCode Command-line Tools from a Disk Image File ( ) to a directory of your choice. This gives you access to all available Command-line Tools, even old ones that would normally not install on a newer version of macOS.

Note. For XCode Disk Image Files from version 8.0 and higher, pbzx is needed and will be downloaded if not available. The location that XCode CLTools Installer will be installed to via PPYM must be writable by the user running it (which it usually is when using ) or otherwise needs to be called by the user that installed it with .|||
