Nodemailer is a module for Node.js applications to allow easy as cake email sending. The project got started back in 2010 when there was no sane option to send email messages, today it is the solution most Node.js users turn to by default.

The current version of Nodemailer v4+ is licensed under MIT license. See license details in the License page. If you are upgrading from Nodemailer v2 or older, then see the light migration guide here.

Also check out the Wild Duck IMAP server, it is built on the same foundation as Nodemailer and provides an open source distributed email server that uses MongoDB as email store. Wild Duck IMAP Server

If you are able to run Node.js version 6 or newer, then you can use Nodemailer. There are no platform or resource specific requirements.
