A complete, pure JavaScript, streamed JSON parser in about (gzipped). It is similar to Oboe.js, a streaming JSON micro-library with a size of (gzipped). While that alone isn't much, sizes add up quickly when you stack many libs. This lib achieves a 85% size reduction, while still offering the same main functionality. Uses ES6 arrows.

This library has no tests yet and could contain buggy edge-cases.|||
