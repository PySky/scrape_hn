Aspose.Diagram for .NET and Java are native APIs that give you full control working with Microsoft Visio files. It allows developers to work with all common Visio formats, it allows you to open files and manipulate the elements of the diagram, from lines and fills, to more complex elements, and then export to VDX, VSX, VTX, XPS, HTML, SVG, SWF, XAML, image or PDF formats.
