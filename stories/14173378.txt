An ES6 small library for easy sorting and filtering of elements.

Sortboard is small library for sorting and filtering HTML elements which uses CSS3 and transform functions. It supports RegEx string filters and it's responsive by default.

The UMD build is also available on unpkg.

You can use the library via

Define some list. For example an with child elements, then set your filters in each element with the attribute. It can add several filters by element separated by whitespace.
