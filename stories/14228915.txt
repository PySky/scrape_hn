If you have enabled GitHub integration in the ide, just a caution that you have granted Smart things unrestricted access to read and write all your public and private repo on your GitHub account.

I was never comfortable with this, and disabled my integration immediately. The upside is the ide is much faster without it.

If you develop or use private repos, I highly suggest you remove the SmartThings GitHub integration immediately.|||
